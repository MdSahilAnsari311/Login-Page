# Login Page UI

This project showcases a clean and modern Login Page built using HTML and CSS, with additional UI enhancements using Bootstrap and Font Awesome.
The design is responsive and includes a background image, styled input fields, social login icons, and a polished layout suitable for real-world web applications.

# Features
- Responsive Login UI
- Clean and modern design
- Email & password input fields
- “Remember Password” checkbox
- Social sign-in icons (Google, Facebook, Twitter)
- Background image with blur-glass effect
- Mobile-friendly layout

# Technologies Used
- HTML5
- CSS3
- Bootstrap 5 (for form styling)
- Font Awesome (for social icons)

# UI Preview
- The page includes:
- A centered login form
- Glass-effect container
- Styled submit button
- Signup redirect link
- Social login section

# How to Use
- Clone or download the project
- Place all files in the same folder (HTML, CSS, images)
- Open index.html / login.html in your browser
- Customize styles or form actions as needed
